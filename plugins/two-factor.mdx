`OTP` `TOTP` `Backup Codes` `Trusted Devices`

Two-Factor Authentication (2FA) adds an extra security step when users log in. Instead of just using a password, they'll need to provide a second form of verification. This makes it much harder for unauthorized people to access accounts, even if they've somehow gotten the password.

This plugin offers two main methods to do a second factor verification:

1. **OTP (One-Time Password)**: A temporary code sent to the user's email or phone.
2. **TOTP (Time-based One-Time Password)**: A code generated by an app on the user's device.

**Additional features include:**
- Generating backup codes for account recovery
- Enabling/disabling 2FA
- Managing trusted devices

## Installation

<Steps>
  <Step>
    ### Add the plugin to your auth config

    Add the two-factor plugin to your authentication configuration and specify your app name as the issuer.

    ```ts title="auth.ts"
    import { betterAuth } from "better-auth"
    import { twoFactor } from "better-auth/plugins"

    export const auth = betterAuth({
      appName: "My App", // provide your app name. It'll be used as an issuer.
      plugins: [
        twoFactor()
      ]
    })
    ```
  </Step>
  
  <Step>
    ### Migrate the database

    Run the migration or generate the schema to add the necessary fields and tables to the database.

    <Tabs items={["migrate", "generate"]}>
      <Tab value="migrate">
      ```bash
      npx auth migrate
      ```
      </Tab>
      <Tab value="generate">
      ```bash
      npx auth generate
      ```
      </Tab>
    </Tabs>
  </Step>

  <Step>
    ### Add the client plugin

    Add the client plugin and specify where the user should be redirected if they need to verify 2nd factor.

    ```ts title="auth-client.ts"
    import { createAuthClient } from "better-auth/client"
    import { twoFactorClient } from "better-auth/client/plugins"

    export const authClient = createAuthClient({
      plugins: [
        twoFactorClient()
      ]
    })
    ```
  </Step>
</Steps>

## Usage

### Enabling 2FA

To enable two-factor authentication, call `twoFactor.enable` with the user's password:

```ts
import { authClient } from "@/lib/auth-client"

const { data } = await authClient.twoFactor.enable({
  password: "user-password"
})

if (data) {
  // Save backup codes securely
  console.log(data.backupCodes)
  // Display QR code for TOTP
  console.log(data.totpURI)
}
```

When 2FA is enabled:
- An encrypted `secret` and `backupCodes` are generated
- `enable` returns `totpURI` and `backupCodes`
- `twoFactorEnabled` won't be set to `true` until the user verifies their TOTP code

<Callout type="warn">
Two Factor can only be enabled for credential accounts at the moment. For social accounts, it's assumed the provider already handles 2FA.
</Callout>

### Sign In with 2FA

When a user with 2FA enabled tries to sign in, the response will contain `twoFactorRedirect` set to `true`:

```ts
import { authClient } from "@/lib/auth-client"

await authClient.signIn.email({
  email: "user@example.com",
  password: "password123",
}, {
  async onSuccess(context) {
    if (context.data.twoFactorRedirect) {
      // Redirect to 2FA verification page
      window.location.href = "/verify-2fa"
    }
  },
})
```

Or configure it globally:

```ts title="auth-client.ts"
import { createAuthClient } from "better-auth/client"
import { twoFactorClient } from "better-auth/client/plugins"

const authClient = createAuthClient({
  plugins: [
    twoFactorClient({
      onTwoFactorRedirect() {
        window.location.href = "/verify-2fa"
      },
    }),
  ],
})
```

### TOTP Verification

After enabling 2FA, display the TOTP URI as a QR code:

```tsx
import { authClient } from "@/lib/auth-client"
import QRCode from "react-qr-code"

export default function TwoFactorSetup() {
  const [totpURI, setTotpURI] = useState("")
  
  const getTOTPURI = async () => {
    const { data } = await authClient.twoFactor.getTotpUri({
      password: "user-password"
    })
    if (data) {
      setTotpURI(data.totpURI)
    }
  }
  
  return (
    <div>
      {totpURI && <QRCode value={totpURI} />}
    </div>
  )
}
```

Verify the TOTP code:

```ts
import { authClient } from "@/lib/auth-client"

const { data } = await authClient.twoFactor.verifyTotp({
  code: "123456",
  trustDevice: true // Optional: remember this device for 30 days
})
```

### OTP via Email

Configure OTP sending in your Better Auth instance:

```ts title="auth.ts"
import { betterAuth } from "better-auth"
import { twoFactor } from "better-auth/plugins"

export const auth = betterAuth({
  plugins: [
    twoFactor({
      otpOptions: {
        async sendOTP({ user, otp }, ctx) {
          // Send OTP to user's email
          await sendEmail({
            to: user.email,
            subject: "Your verification code",
            body: `Your code is: ${otp}`
          })
        },
      },
    })
  ]
})
```

Send and verify OTP:

```ts
import { authClient } from "@/lib/auth-client"

// Send OTP
await authClient.twoFactor.sendOtp()

// Verify OTP
const { data } = await authClient.twoFactor.verifyOtp({
  code: "123456",
  trustDevice: true
})
```

### Backup Codes

Generate backup codes for account recovery:

```ts
import { authClient } from "@/lib/auth-client"

const { data } = await authClient.twoFactor.generateBackupCodes({
  password: "user-password"
})

if (data) {
  // Display backup codes to user
  console.log(data.backupCodes)
}
```

Verify a backup code:

```ts
import { authClient } from "@/lib/auth-client"

const { data } = await authClient.twoFactor.verifyBackupCode({
  code: "backup-code-here"
})
```

<Callout type="warn">
Once a backup code is used, it will be removed from the database and can't be used again.
</Callout>

### Trusted Devices

Mark a device as trusted to skip 2FA for 30 days:

```ts
const { data } = await authClient.twoFactor.verifyTotp({
  code: "123456",
  trustDevice: true // Device will be trusted for 30 days
})
```

### Disabling 2FA

To disable two-factor authentication:

```ts
import { authClient } from "@/lib/auth-client"

const { data } = await authClient.twoFactor.disable({
  password: "user-password"
})
```

## Options

### Server Options

```ts title="auth.ts"
import { betterAuth } from "better-auth"
import { twoFactor } from "better-auth/plugins"

export const auth = betterAuth({
  plugins: [
    twoFactor({
      // Custom issuer name (defaults to appName)
      issuer: "My App",
      
      // Skip verification before enabling 2FA
      skipVerificationOnEnable: false,
      
      // TOTP options
      totpOptions: {
        digits: 6,
        period: 30, // seconds
      },
      
      // OTP options
      otpOptions: {
        async sendOTP({ user, otp }, ctx) {
          // Send OTP implementation
        },
        period: 3, // minutes
        storeOTP: "plain", // or "encrypted" or "hashed"
      },
      
      // Backup code options
      backupCodeOptions: {
        amount: 10,
        length: 10,
        storeBackupCodes: "plain", // or "encrypted"
      },
    })
  ]
})
```

### Client Options

```ts title="auth-client.ts"
import { createAuthClient } from "better-auth/client"
import { twoFactorClient } from "better-auth/client/plugins"

const authClient = createAuthClient({
  plugins: [
    twoFactorClient({
      onTwoFactorRedirect() {
        // Handle 2FA redirect
        window.location.href = "/verify-2fa"
      }
    })
  ]
})
```

## Schema

The plugin adds the following to your database:

### User Table

- `twoFactorEnabled` (boolean): Whether 2FA is enabled

### Two Factor Table

- `id` (string): Unique identifier
- `userId` (string): Foreign key to user
- `secret` (string): Encrypted TOTP secret
- `backupCodes` (string): Encrypted backup codes
