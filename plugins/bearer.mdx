---
title: Bearer Token Authentication
description: Authenticate API requests using Bearer tokens instead of browser cookies
---

The Bearer plugin enables authentication using Bearer tokens as an alternative to browser cookies. It's designed for APIs that don't support cookies or require Bearer token authentication.

<Callout type="warn">
Use this cautiously; it's intended only for APIs that don't support cookies. Improper implementation could lead to security vulnerabilities.
</Callout>

## Installation

Add the Bearer plugin to your authentication setup:

```ts title="auth.ts"
import { betterAuth } from "better-auth"
import { bearer } from "better-auth/plugins"

export const auth = betterAuth({
  plugins: [
    bearer()
  ]
})
```

## Usage

### 1. Obtain the Bearer Token

After sign-in, store the session token from the response headers:

```ts
import { authClient } from "@/lib/auth-client"

const { data } = await authClient.signIn.email({
  email: "user@example.com",
  password: "securepassword"
}, {
  onSuccess: (ctx) => {
    const authToken = ctx.response.headers.get("set-auth-token")
    localStorage.setItem("bearer_token", authToken)
  }
})
```

Or configure it globally:

```ts title="auth-client.ts"
import { createAuthClient } from "better-auth/client"

export const authClient = createAuthClient({
  fetchOptions: {
    onSuccess: (ctx) => {
      const authToken = ctx.response.headers.get("set-auth-token")
      if (authToken) {
        localStorage.setItem("bearer_token", authToken)
      }
    }
  }
})
```

### 2. Configure the Auth Client

Set up your client to include the Bearer token in all requests:

```ts title="auth-client.ts"
import { createAuthClient } from "better-auth/client"

export const authClient = createAuthClient({
  fetchOptions: {
    auth: {
      type: "Bearer",
      token: () => localStorage.getItem("bearer_token") || ""
    }
  }
})
```

### 3. Make Authenticated Requests

Now all requests are automatically authenticated:

```ts
import { authClient } from "@/lib/auth-client"

const { data } = await authClient.listSessions()
```

### 4. Per-Request Token (Optional)

Provide token for individual requests:

```ts
import { authClient } from "@/lib/auth-client"

const { data } = await authClient.listSessions({
  fetchOptions: {
    headers: {
      Authorization: `Bearer ${token}`
    }
  }
})
```

### 5. Using Bearer Tokens Outside Auth Client

Use the token with any API request:

```ts
const token = localStorage.getItem("bearer_token")

const response = await fetch("https://api.example.com/data", {
  headers: {
    Authorization: `Bearer ${token}`
  }
})

const data = await response.json()
```

### Server-Side Authentication

Authenticate requests on the server:

```ts title="session.ts"
import { auth } from "@/lib/auth"

export async function handler(req, res) {
  // Requires Authorization Bearer token header
  const session = await auth.api.getSession({
    headers: req.headers
  })
  
  if (!session) {
    return res.status(401).json({ error: "Unauthorized" })
  }
  
  // Process authenticated request
}
```

## Options

**requireSignature** (boolean): Require the token to be signed. Default: `false`.

```ts title="auth.ts"
import { betterAuth } from "better-auth"
import { bearer } from "better-auth/plugins"

export const auth = betterAuth({
  plugins: [
    bearer({
      requireSignature: true
    })
  ]
})
```

## Best Practices

1. **Secure Storage**: Store tokens securely (avoid localStorage for sensitive apps)
2. **Token Expiration**: Implement token refresh logic
3. **HTTPS Only**: Always use HTTPS in production
4. **Clear on Logout**: Remove tokens when users sign out
5. **Validate Server-Side**: Always verify tokens on the server

## Example: Complete Implementation

```ts title="auth-client.ts"
import { createAuthClient } from "better-auth/client"

const TOKEN_KEY = "auth_token"

export const authClient = createAuthClient({
  fetchOptions: {
    auth: {
      type: "Bearer",
      token: () => localStorage.getItem(TOKEN_KEY) || ""
    },
    onSuccess: (ctx) => {
      const token = ctx.response.headers.get("set-auth-token")
      if (token) {
        localStorage.setItem(TOKEN_KEY, token)
      }
    },
    onError: (ctx) => {
      if (ctx.response.status === 401) {
        localStorage.removeItem(TOKEN_KEY)
        window.location.href = "/login"
      }
    }
  }
})

// Sign out helper
export async function signOut() {
  await authClient.signOut()
  localStorage.removeItem(TOKEN_KEY)
}
```
